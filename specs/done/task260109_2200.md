# 작업 계획서: 비슷한 얼굴 찾기 기능을 별도 패널로 분리

**작업 일시**: 2026-01-09 22:00  
**작업 목표**: 비슷한 얼굴 찾기 기능을 얼굴 추출 패널에서 분리하여 독립적인 패널로 구현

## 작업 배경
- 현재 비슷한 얼굴 찾기 기능이 얼굴 추출 패널 안에 포함되어 있음
- 별도 패널로 분리하여 독립적으로 사용할 수 있도록 개선 필요
- 메뉴에서 직접 접근할 수 있도록 추가

## 작업 순서

### 1단계: 새로운 비슷한 얼굴 패널 파일 생성
- [x] `gui/_face_similar.py` 새 파일 생성
  - `FaceSimilarPanel` 클래스 구현 (tk.Toplevel 상속)
  - `SimilarFaceManagerMixin` 사용
  - 독립적인 UI 구성
  - 파일 선택, 검색, 결과 표시 기능

### 2단계: 얼굴 추출 패널에서 비슷한 얼굴 UI 제거
- [x] `gui/_face_extract.py`에서 비슷한 얼굴 관련 코드 제거
  - `_create_similar_faces_ui()` 함수 제거
  - `on_find_similar_faces()` 함수 제거
  - `_display_similar_faces()` 함수 제거
  - `_load_similar_face_image()` 함수 제거
  - `SimilarFaceManagerMixin` 제거 (Mixin 목록에서)
  - 관련 변수들 제거

### 3단계: 비슷한 얼굴 패널 UI 구현
- [x] 파일 선택 UI
  - 이미지 파일 목록 표시
  - 현재 선택된 이미지 표시
- [ ] 검색 UI
  - "비슷한 얼굴 찾기" 버튼
  - 검색 진행률 표시
- [ ] 결과 표시 UI
  - 스크롤 가능한 목록
  - 썸네일, 파일명, 유사도 점수
  - 클릭 시 해당 이미지 선택

### 4단계: 메뉴에 추가
- [x] `gui/gui.py`에 메뉴 항목 추가
  - "비슷한 얼굴 찾기..." 메뉴 항목
  - `show_face_similar_panel()` 함수 호출

## 구현 세부사항

### 비슷한 얼굴 패널 구조
- 왼쪽: 파일 목록 (리스트박스)
- 오른쪽 상단: 현재 선택된 이미지 미리보기
- 오른쪽 하단: 비슷한 얼굴 목록 (검색 결과)

### 기능
- 파일 목록에서 이미지 선택
- 선택한 이미지와 비슷한 얼굴 검색
- 검색 결과 목록 표시
- 결과 항목 클릭 시 해당 이미지 선택
